Очередь выполненения скриптов:
1. скопируйте скрипт из файла 1_ISP.txt в созданный вами txt файл на ISP, сконфигурируйте и запустите
2. скопируйте команды из 2_HQ-RTR.txt в консоль HQ-RTR
3. скопируйте команды из 3_BR-RTR.txt в консоль BR-RTR
4. скопируйте скрипт из файла 4_HQ-SRV.txt в созданный вами txt файл на HQ-SRV, сконфигурируйте и запустите
5. скопируйте скрипт из файла 5_BR-SRV.txt в созданный вами txt файл на BR-SRV, сконфигурируйте и запустите
6. скопируйте скрипт из файла 6_HQ-CLI.txt в созданный вами txt файл на HQ-CLI и запустите
7. скопируйте команды из 7_HQ-RTR.txt в консоль HQ-RTR
8. скопируйте команды из 8_BR-RTR.txt в консоль BR-RTR

Примечание! После каждого шага (завершения скрипта) чистите оставшиеся файлы и историю команд на устройствах (инструкция в 9_Uninstall.txt).

Для исполнения скрипта нужно выполнить несколько действий:
1. выполните apt-get update для обновления списка пакетов (Примечание! Если apt-get update выполняется очень долго, то необходимо сменить источник загрузки. Для этого используйте vim и закоментируйте/раскоментируйте нужные строки в /etc/apt/sources.list.d/alt.list (закоментируйте все 3 строки) и в /etc/apt/sources.list.d/yandex.list (раскоментируйте 3 строки https://mirror...))
2. установите wget и dos2unix используя команду apt-get install -y wget dos2unix
3. c помощью wget скачайте нужный файл скрипта из репозитория используя следующий формат: wget https://raw.githubusercontent.com/DimaDemoExam/<номер модуля>/refs/heads/main/<имя файла>.sh (Пример: wget https://raw.githubusercontent.com/DimaDemoExam/Module1/refs/heads/main/1_ISP.sh)
4. сконвертируйте файл в нужный формат используя dos2unix: dos2unix <имя файла>.sh
5. сконфигурируйте необходимые значения в скрипте используя vim
6. запустите скрипт используя команду bash <имя файла>.sh

Примечение! В некоторых местах скрипта можно поменять значения на необходимые.
